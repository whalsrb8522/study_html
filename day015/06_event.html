document 영역에서 사용되는 이벤트를 정의

마우스, 키보드, 실행 등등 이벤트의 종류가 매우 많음

  • 마우스 이벤트 : click(), focus(), blur(), input(), change(), mouseover(), mouseout(), select(), scroll(), load(), drag() => drop()
  • 키보드 이벤트 : event.key => keyDown(), keyPress(), keyUp()
  • 기타 이벤트 관련 메소드 : preventDefault(), target, submit()
  • addEventListener('eventName', callback Function) 메소드를 사용하여 구현하는 것을 권장
    메소드를 태그 속성이나 ,엘리먼트에 직접 부여하는 방식도 있음
  • addEventListener를 권장하는 이유는 같은 엘리먼트에 여러 이벤트를 중복 부여 할 수 있으며,
    적용된 이벤트를 선택적으로 제거 할 수 있기 때문
