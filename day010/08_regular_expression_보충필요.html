정규식(Regular Expression) : 문자열에서 특정 내용을 찾거나 대체, 발췌하는데 사용

  • regexr 단어로 많이 표현됨
  • 특정 데이터를 탐색, 확인, 존재여부 등을 쉽게 확인하기 위한 규칙
  • 웹 분야에서는 password 작성 규칙, 첨부파일 ... 자주 사용하지는 않음
  • 데이터 분석 관련 분야에서는 반즈시 알고 있어야 함
  • /regexr/i :
    • (//) : 시작, 종료기호
    • (regexr) : pattern
    • (i) : flag
  • ("str").match(/regexr/flag): str 문자열에서 매칭되는 항목들을 배열로 반환
  • ("str").replace(/regexr/flag, "바꿀문자열"):
  • ("str").split(/regexr/flag): 문자열을 해당되는 항목으로 쪼개어 배열로 반환
  • (정규표현식).text("str"): 문자열이 정규표현식 규칙에 맞으면 true, 아니면 false
  • 정규표현식 플래스
  • i : 대/소문자 구분X
  • g : 전체 검색
  • m : 문자열의 행이 바뀌어도 계속 검색 (mluti-line)
  • a-zA-Z : 영어 알파멧 / ㄱ-ㅎ가-힣 : 한글 / 0-9 : 숫자
  • . : 모든 문자열 (줄바꿈 제외)
  • \d : 숫자 / \D : 숫자 제외
  • \s : 공백 / \S : 공백 제외
  • \w : 영숫자, _ / \W: \w 제외
  • 검색패턴 : | or a|b
  • [] : 괄호 내 문자들 중 하나
  • /abc/ : "abc"를 포함하는
  • /[abc] : "a" 또는 "b" 또는 "c"를 포함하는
  • [다-바] : 다, 라, 마, 바 중 하나 이상을 포함하는
  • [^문자] : 괄호안의 문자를 제외한 것
  • [^abcd] : "a" "b" "c" "d" 4개 문자를 제외
  • ^문자열$ : ^(시작) $(끝)
