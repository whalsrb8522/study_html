배열 관련 메소드: 유사(연관) 배열에서는 사용 제한이 있음.

  • join(): 배열 값 사이에 문자를 삽입하여 새로운 문자열 생성
  • array.push(value): 배열의 마지막에 value값을 추가 == array(array.length) = value;, 배열의 길이를 리턴
  • array.pop(): 배열의 마지막 값 제거, 삭제된 값을 리턴
  • array.shift(): 배열의 처음 값 제거, 삭제된 값을 리턴
  • array.unshift(value): 배열의 가장 앞에 값을 추가, 추가 후 배열 길이를 리턴
