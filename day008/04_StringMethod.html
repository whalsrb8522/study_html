가장 많이 다루는 데이터 타입 : String

  • 문자열 데이터는 index(0부터 시작)을 가지고 있음
  • 문자열 데이터 값 내에서는 개행(br) 하는 것을 추천하지 않는다.
  • new String(); 객체 타입으로 선언하여 사용가능하지만, 추천하지 않는다.
  • length: 문자열의 길이 => index는 length - 1

문자 관련 메소드: java에서 사용하는 String 메소드와 비슷

  • 문자 추출 관련 메소드: substring(), substr(), slice()
  • 문자 대체 메소드: replace(oldStr, newStr): 첫번쨰 매칭되는 문자만, 대소문자 구분
  • trim(): 문자열 양측 공백 제거
  • padStrart(목표문자길이, 채울문자) / padEnd(목표문자길이, 채울문자)
    앞쪽에서 / 뒷쪽에서 채우기
  • charAt(index): 해당 index 문자 추춘
  • indexOf("str"): 문자열의 위치를 반환
  • startWith("str") / endWith("str"): 지정 문자열이 탐색대상 문자열에서 시작/끝에 존재하는지 확인하는 메소드
