DOM(Document Object Model) : 문서(html) 객체 모델

  • html의 하위 태그는 각각의 기능(Method), 속성(Properties)를 가지고 있는 문서 객체
  • 엘리먼트 가져오기 : tag element로 가져오는 메소드 byId, byClassName, byTagName
  • document.getEelementById('idName') : 단일 엘리먼츠 리턴
  • document.getEelementByClassName('className') : 복수 엘리먼츠 리턴 -> HTMLCollection 타입으로 리턴
  • document.getEelementByTagName('tagName') : 복수 엘리먼츠 리턴 -> HTMLCollection 타입으로 리턴
  • getElements로 가져오는 객체는 우선적으로 HTMLCollection의 index를 선택해야 함
