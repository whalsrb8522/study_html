자바스크립트의 데이터 타입의 확장 개념 집합데이터를 다룰 수 있는 객체

  • set은 중복 데이터 허용 안함 (중복 데이터가 들어올 경우 체크 후 하나만 남김)
  • set() : 파라미터는 주로 배열을 사용, 배열의 index, value 값을 활용하여 집합데이터를 만듬
  • set 전용 메소드들이 있기 때문에 별도 학습 필요
  • new set() or set(array) : set 객체의 생성 및 초기화
  • set == s, s.add(value) : 데이터 추가
  • s.delete(value) : 데이터 삭제
  • s.has(value) : 데이터의 존재여부를 true/false로 리턴
  • s.forEach(callback function) : set 순환
  • s.entries(), s.keys(), s.values() : 리턴타입은 setIterator, 별도의 메소드를 사용하여 접근
  • s.size : 전체 set의 개수
